# 📊 E-commerce Sales Dashboard (Power BI)

## 📌 Overview

This project presents an interactive Power BI dashboard built to analyze e-commerce sales performance. It includes key metrics, trend analysis, customer insights, and geographic distribution.

---

## 🎯 Objective

* Track business KPIs
* Analyze revenue trends
* Understand customer behavior
* Identify top-performing products and categories

---

## 📊 Dashboard Highlights

### 🔢 Key Metrics

* Total Revenue: **$91.83M**
* Total Orders: **100K**
* Total Customers: **43K**
* Avg Order Value: **$918.26**

---

### 📈 Sales Analysis

* Monthly revenue trend across multiple years
* Seasonal patterns identified

---

### 🛍️ Product & Category Insights

* Top 5 revenue-generating products
* Profit analysis by category
* High-performing categories identified

---

### 💳 Payment Analysis

* Distribution of payment methods
* Majority transactions via digital payments

---

### 🚚 Order Status

* Majority orders successfully delivered
* Low cancellation and return rates

---

### 🌍 Geographic Insights

* Sales distribution across different cities
* Identifies high-performing regions

---

## ⚙️ Data Modeling & Techniques

* Created **Date Table** for time intelligence
* Built **DAX measures** for KPIs
* Created **calculated columns** for better insights
* Applied data transformations

---

## 🛠 Tech Stack

* Power BI
* DAX
* Data Modeling
